Для ТМ и ЛС сделаны доп. переключатели для нормальной работы по RS-422. 
Переделать дискретные входы ?!

Стэк:
18мкм  TOP 
1.5мм e=4.5 FR-4
18мкм  BOTTOM


03v3
Предназначена для использования в ОПТИКЕ и АВАНТах К400, РЗСК, Р400М 
сделанными в новом варианте.
Используется совместно с BVP не ранее 04v2, если не нужна цифровая
ретрансляция. И не ранее 04v3 если нужна.

Изменения:
Переделана цивровая ретрансляция. Сделан чистый 422 интерфейс и добавлен
сигнал управления.
Терминаторы 120 и 51Ом изменены на 240Ом.

Основной вариант (Р400М):
отсутствуют: цифровая ретрансляция, выходы ком.ПРД/ПРМ, вход ПУСК.

Вариант Р400:
добавлен выход "Предупреждение"

Вариант К400 (РЗСК):
плата запаивается полностью.

Недостатки:
Резисторы подтяжки стоят на выводах X/Y, что при переключении на 422 убирает
подтяжки с выводов A/B и оставляет вход 422 интерфейса без подтяжки. 
Вообще дуплекс/полудуплекс сделан хреново.
Место резисторов можно поставить предохранители.


!!!
На данный момент входы управления запаиваются только под 24.
Номиналы резисторов для других напряжений см. в РЭ.
